# PokeDex - A Nuxt 3 Project

PokeAPI と TypeScript を使用して作成した、ポケモン図鑑（PokeDex）の実用的なサンプルです。

# Zenn

https://www.npmjs.com/package/zod

# 機能

- ポケモンリストページ
  ポケモンの名前・画像・タイプを一覧で表示します。

- ポケモン詳細ページ
  選択したポケモンの詳細情報（ステータスや特性など）を表示します。

# 技術スタック

- Nuxt 3
- PokeAPI：本アプリケーションで使用するポケモンデータの取得元
- TypeScript
- TailwindCSS
